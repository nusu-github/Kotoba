# 言（ことば）言語仕様 v1.0 — 例外処理

> 状態: Draft  
> バージョン: v1.0  
> 関連規範 ID: EXC-001..006

---

## 9.1 構造

**[規範] EXC-001 (MUST)**
`試す` は式である。成功時は本体値、失敗時は `失敗した場合` 値を返す。

**[規範] EXC-002 (MUST)**
`失敗した場合` がないときに例外が起きた場合、`試す` 式の値は `無` とする。

**[規範] EXC-003 (MUST)**
`必ず行う` の評価結果は常に破棄する。

---

## 9.2 例外伝搬

**[規範] EXC-004 (MUST)**
`訴える` は最寄りの `試す` へ例外を送出する。

**[規範] EXC-005 (MUST)**
`必ず行う` 内で再送出した例外は元例外を上書きする。

**[規範] EXC-006 (SHOULD NOT)**
`必ず行う` 内での再送出は避けるべきである。
