## 1.1 適用範囲

**[規範] GEN-001 (MUST)**
本仕様は、言（ことば）言語の字句・構文・意味・型・実行モデル・診断・適合性を定義する。

**[規範] GEN-002 (MUST)**
本仕様における曖昧な自然言語表現は、すべて本章の用語定義と規範キーワードに従って解釈する。

**[参考]**
v1 は v0.4 を凍結参照しつつ、互換より一貫性を優先して再定義する。

## 1.2 用語定義

- `役割助詞`: 手順呼び出しの引数役割を示す助詞（`を/が/に/で/から/まで/と/へ/より`）
- `アクセス助詞`: 属性・環境参照のための助詞（`の`）
- `束縛マーカー`: 束縛を表す助詞（`は`）
- `助詞式`: `(式 役割助詞)+ 動詞` で表す呼び出し式
- `アクセス式`: `式 の 識別子` の左結合連鎖
- `束縛式`: `束縛対象 は 式`（文）
- `て形チェイン`: 暗黙 `を` 受け渡しを伴う処理連鎖
- `環境参照`: `これのX/それのX/あれのX` によるスコープ参照

**[規範] GEN-003 (MUST)**
同一語に複数の意味を与えてはならない。意味が複数候補になる場合は、構文で分岐させる。

---

## 1.3 規範キーワード

**[規範] GEN-004 (MUST)**
`MUST`、`MUST NOT`、`SHOULD`、`SHOULD NOT`、`MAY` を以下の意味で用いる。

| キーワード | 意味       |
| ---------- | ---------- |
| MUST       | 必須要件   |
| MUST NOT   | 禁止要件   |
| SHOULD     | 推奨要件   |
| SHOULD NOT | 非推奨要件 |
| MAY        | 任意要件   |
