# 言（ことば）言語仕様 v1.0 — 制御構造（式意味論）

> 状態: Draft  
> バージョン: v1.0  
> 関連規範 ID: CTR-001..006

---

## 8.1 if 式

**[規範] CTR-001 (MUST)**
`もし` 式は選択された分岐ブロックの最終値を返す。

**[規範] CTR-002 (MUST)**
`もしくは` は上から順に評価し、最初に真となる分岐のみ実行する。

---

## 8.2 match 式

**[規範] CTR-003 (MUST)**
`どれかを調べる` は上から順に最初に一致するアームを実行する。

**[規範] CTR-004 (SHOULD)**
`どれでもない場合` を記述することを推奨する。

---

## 8.3 loop 式

**[規範] CTR-005 (MUST)**
繰り返し式の値は `無` とする。

**[規範] CTR-006 (MUST)**
`次へ` と `抜ける` はループ文脈内でのみ使用できる。
