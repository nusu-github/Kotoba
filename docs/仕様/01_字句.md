# 言（ことば）言語仕様 v1.0 — 字句仕様

> 状態: Draft  
> バージョン: v1.0  
> 関連規範 ID: LEX-001..012

---

## 2.1 トークンカテゴリ

**[規範] LEX-001 (MUST)**
字句解析器は次のカテゴリを区別する。

- リテラル（整数、小数、文字列、真偽、無）
- 識別子
- 助詞（役割助詞、アクセス助詞、束縛マーカー）
- キーワード
- 記号（`【】｛｝（）`、`、`、`。`、`:`、`…`）
- 構造トークン（`INDENT`、`DEDENT`、`SEP`、`EOF`）

---

## 2.2 文区切り・インデント

**[規範] LEX-002 (MUST)**
改行と `。` は等価な文区切りとして `SEP` に正規化する。

**[規範] LEX-003 (MUST)**
同一ファイル内でタブとスペースの混在インデントを禁止する。

**[規範] LEX-004 (MUST)**
`、` はて形チェイン内部の区切りにのみ使用する。

---

## 2.3 助詞分離

**[規範] LEX-005 (MUST)**
識別子末尾に助詞が付着した場合、最長一致で分離する。

例:

- `名前を` -> `識別子(名前)` + `助詞(を)`
- `地点まで` -> `識別子(地点)` + `助詞(まで)`
- `比較より` -> `識別子(比較)` + `助詞(より)`

**[規範] LEX-006 (MUST NOT)**
`の` と `は` は助詞式の引数助詞として扱ってはならない。

---

## 2.4 キーワード優先

**[規範] LEX-007 (MUST)**
キーワード完全一致は助詞分離より優先する。

例:

- `そうでなければ` は単一キーワード
- `どれかを調べる` は単一キーワード

---

## 2.5 数値と助数詞

**[規範] LEX-008 (MUST)**
`数値 + 助数詞` は単一の助数詞付き数値として字句化する。

**[規範] LEX-009 (MUST)**
助数詞の字種は漢字・カタカナのみを許可し、ひらがなを禁止する。

例:

- `5秒` は有効
- `100ピクセル` は有効
- `2の` は `整数(2)` + `アクセス助詞(の)`

---

## 2.6 予約語

**[規範] LEX-010 (MUST)**
以下は予約語であり、識別子として使用してはならない。

| 区分       | 予約語                                                               |
| ---------- | -------------------------------------------------------------------- |
| 制御       | `もし` `ならば` `もしくは` `そうでなければ` `次へ` `抜ける`          |
| 例外       | `試す` `失敗した場合` `必ず行う` `訴える`                            |
| 定義       | `という` `手順` `組` `特性` `公開`                                   |
| 束縛       | `は` `変わる` `変える` `返す`                                        |
| モジュール | `使う`                                                               |
| パターン   | `どれかを調べる` `の場合` `どれでもない場合` `どれでもない` `どれか` |
| 連鎖       | `分岐して`                                                           |
| 論理       | `かつ` `または` `でない`                                             |
| こそあど   | `これ` `それ` `あれ` `こう` `ここ` `そこ`                            |
| 将来予約   | `しながら` `待つ` `背景で`                                           |

**[規範] LEX-011 (MUST)**
`しながら`、`待つ`、`背景で` は v1 では未実装予約語とし、使用時は未実装エラーとする。

---

## 2.7 字句エラー

**[規範] LEX-012 (MUST)**
字句エラーは行・列・スパンとともに報告し、以降の処理を継続可能な最短回復を行う。
