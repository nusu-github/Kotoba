# 言（ことば）v1 適合テスト台帳

> 対象仕様: [言語仕様_v1.md](/home/nusu/Kotoba/docs/言語仕様_v1.md)
> 追跡キー: ケース ID（ユニーク）

---

## 1. 台帳運用規則

- すべてのケースは `受理` または `拒否` を持つ。
- `拒否` ケースは、期待診断（要点）を記載する。
- ケース ID は再利用しない。
- 規範ルール ID へのリンクは複数指定してよい。
